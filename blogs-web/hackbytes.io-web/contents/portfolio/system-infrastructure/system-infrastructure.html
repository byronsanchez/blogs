Project Details

System Infrastructure is the configuration management project used for all nodes on the nitelite.io network. Whenever a new node is provisioned, the configurations defined in this project are applied to the node so that the node can be up and running as soon as possible. This project uses Puppet for configuration management. The nodes use Gentoo Linux for the operating system.
