Project Details

hackBytes.io is the site I run for software, productivity, and other engineering and technical projects. I sometimes like to write about anything related to the bleeding edge of product development and productivity, so hackBytes.io is my channel for these writings.

hackBytes.io replaces my old hackbytes.com blog.

Built with powerful systems.

This entire website is built on top of and makes use of some of the most powerful systems available in web application development.

Statically-generated

Almost every part of the site is statically generated. This means that most of what a user sees are previously generated html files (thanks to Wintersmith). This is a perfect system for a simple blog, where dynamic programs typically cause much more overhead than it's worth.

The portions of the site that make use of dynamic content generation use CoffeeScript (to JavaScript) to PHP as the client to server languages. Once the server has generated tailored content, it passes the generated data back to the JavaScript file as a JSON object. From there, the data is presented to the user.

Powered by Foundation

The website is built on top of Zurb's Foundation Framework. The website is mobile-ready and works on a wide range of devices.
